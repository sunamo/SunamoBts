### SunamoBts

Advanced utilities for working with base .NET types including parsing, conversion, and type system operations.

Part of PlatformIndependentNuGetPackages:

- [nuget.org](https://www.nuget.org/profiles/sunamo)
- [github.org](https://github.com/sunamo/PlatformIndependentNuGetPackages)

Another links:

- [Developer site](https://sunamo.cz)

Request for new features / bug report / etc: [Mail](mailto:radek.jancik@sunamo.cz) or on GitHub

## Key Features

- Safe parsing of primitive types (int, float, double, bool, byte, short, DateTime) with default value support
- Type validation methods (IsInt, IsFloat, IsDouble, IsLong, IsBool, IsByte, IsDateTime)
- Boolean/string/integer conversions
- Stream and byte array utilities
- Collection-to-number conversion with length validation

## Target Frameworks

**TargetFrameworks:** `net10.0;net9.0;net8.0`
